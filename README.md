# 🍵 Matcha Log

Matcha Log is a simple web app for keeping track of your favorite matcha tins. Add matcha names and ratings, then view them in a clean, sortable list.

![matcha log overview](public/pic.png)

##  Features
- Add matcha entries (name + rating)
- View a list of saved matcha
- Sort by rating or name
- Delete entries
- Data saved using localStorage

##  Built With
- React
- Vite
- Tailwind CSS
- LocalStorage

##  Run the Project

```bash
npm install
npm run dev
```
